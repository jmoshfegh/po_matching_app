# PO Matching System

Three-way Purchase Order (PO) matching desktop-style web application built with Python and Streamlit.

This project performs three-way matching between Purchase Orders (PO), Goods Receipt Notes (GRN), and Invoices (INV), with automatic parameter detection, discrepancy analysis, interactive review, and rich reporting.

Implementation details follow the step-by-step specification in the Cursor prompt and are organized into `core`, `ui`, `data`, `db`, and `tests` packages.
